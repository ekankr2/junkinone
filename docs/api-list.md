### 날씨 API

**설명**: 전국 날씨 정보 조회

**엔드포인트**: `GET /api/weather`

**개발자**: 김철수 ([@kimcs](https://github.com/kimcs))  
**출처**: 웹 3기  
**원본**: [github.com/kimcs/weather-api](https://github.com/kimcs/weather-api)

**사용 예시**:
```bash
curl https://junkinone.com/api/weather?city=서울
```

**응답**:
```json
{
  "city": "서울",
  "temperature": 5.2,
  "status": "맑음"
}
```

**통계**: 
- 일 평균 호출: 487회
- 에러율: 0.1%
